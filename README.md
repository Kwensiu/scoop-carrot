# Welcome!

This is a Scoop Bucket repository, currently only used for storing the following software:

- [Pailer](https://github.com/Kwensiu/Pailer)
- [Rscoop-k](https://github.com/Kwensiu/Rscoop)
- [KeyStats](https://github.com/debugtheworldbot/keyStats)

You can use the following command to add the Bucket:

```
scoop bucket add carrot https://github.com/Kwensiu/scoop-carrot
```

Then use the following command to install the app:

```
scoop install pailer
or
scoop install carrot/pailer
```
